# Organizator_Turystyki
Cel Projektu

Aplikacja z interfejsem graficznym stworzona jako dowód przejścia na ścieżkę Junior Python Developera. Projekt demonstruje zdolność do przeniesienia logiki biznesowej i zarządzania procesami.

Aplikacja symuluje system do zarządzania rezerwacjami, danymi klientów i monitoringiem płatności.

Kluczowe Funkcjonalności Biznesowe

1.Rejestracja Klienta: Wprowadzanie danych, w tym paszportowych (symulacja wymagań RODO).
2.Logika Cenowa: Kalkulacja ceny końcowej na podstawie ceny bazowej i dynamicznych rabatów (last/first minute).
3.Aktualizacja Płatności: Możliwość zmiany statusu (np. "Zapłacono przelewem", "Zapłacono gotówką") i wprowadzanie numerów faktur.
4.Raportowanie: Generowanie listy rezerwacji dla danego terminu wycieczki, weryfikacja stanu płatności oraz raportów analitycznych (z wykorzystaniem SQL).
